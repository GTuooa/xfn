## 用处
点击导出按钮或pdf导出按钮后跳出的选择框，可以选择立即下载（下载至本地）或以钉钉消息形式推送


## API
属性说明如下：

属性           |  说明     |  类型  | 默认值
--------------|----------|-------|------
ddCallback    | 通过钉钉选人组件获取到接收者名字列表后要调用到方法 | function | 无
exportDisable | 导出不能使用时设为true | bool | false
hrefUrl       | 立即下载到下载地址 | string | 无
